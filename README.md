# Tic Tac Toe Game

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## How to use

1. Download or clone this repository.
2. Open `index.html` in your web browser.
3. Play the game!

## Features

- Two-player gameplay
- Simple and clean UI
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
